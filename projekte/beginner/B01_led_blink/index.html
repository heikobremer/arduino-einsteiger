Schritt 2: Blinken mit delay()

// Schritt 2: LED blinken lassen

const int LED_PIN = 7;

void setup() {
  pinMode(LED_PIN, OUTPUT);
  Serial.begin(9600);
  Serial.println("LED Blink-Programm gestartet!");
}

void loop() {
  digitalWrite(LED_PIN, HIGH);  // LED an
  delay(500);                   // 500ms = 0.5s warten

  digitalWrite(LED_PIN, LOW);   // LED aus
  delay(500);                   // 500ms warten
}

Was passiert?

Die loop()-Funktion wird endlos wiederholt. Die LED geht an (0.5s) ‚Üí aus (0.5s) ‚Üí an ... und so weiter!
